## 1) Project Overview

**ASI‑T2** synthesizes, predicts, and optimizes complex aerospace outcomes by leveraging the **TFA (Quantum/Classical)** architecture. This INTELLIGENCE package defines the **PLM layer as Agentic Skills & Competences** operated by T2 agents and Human Intelligence (HI) operators.

*   **Mission**: verifiable, physics‑based governance across aerospace programs (ICAO LTAG, ReFuelEU, EU ETS compatible), with full QS→…→QB traceability.
*   **MAL (Master Application Layer/Logic)**: the "PLC of the domain" orchestrating services, policies, ethics (MAL‑EEM), and CI/CD gates.
*   **DOMAINS Definition (Canon)**: **DOMAINS represent areas of theoretical knowledge and specialization** (ontologies, methods, models, standards, playbooks). They are **not** project/program deliverables; instead they **inform** and **govern** those deliverables via CAx skills and MAP services.
