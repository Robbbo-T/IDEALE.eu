## Sub-System Naming Convention

Format: `[ATA]-[SUBSYSTEM-NUMBER]-[DESCRIPTIVE-NAME]/`  
Example: `53-10-CENTER-BODY/`

- `53` = ATA chapter  
- `10` = Subsystem identifier (typically increments by 10)  
- `CENTER-BODY` = Descriptive name (uppercase, hyphenated)
